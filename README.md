# BlockOracle

BlockOracle is a decentralized prediction game built on Bitcoin L1 using Discreet Log Contracts (DLCs) and Taproot.

Bet in sats.  
No custodians.  
No platform risk.  
Only Bitcoin.

---

## 🔥 Vision

Prediction markets secured directly by Bitcoin.

BlockOracle enables users to forecast real-world events and stake BTC in trust-minimized contracts settled via DLCs.

---

## ⚙️ How It Works

1. A market is created (e.g. BTC weekly close).
2. Users lock sats into a DLC contract.
3. An oracle signs the event outcome.
4. Funds settle on-chain according to the signed result.

No escrow.  
No synthetic tokens.  
Native Bitcoin settlement.
