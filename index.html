// ─── ИСПРАВЛЕННАЯ ФУНКЦИЯ ПОДКЛЮЧЕНИЯ КОШЕЛЬКА ────────────────────────────── async function connectWallet() { const opw = getOPWallet(); if (!opw) { alert(`OPWallet не обнаружен! ${short}`; btn.className = 'btn-conn on'; btn.onclick = null; document.getElementById('btnDisc').style.display = 'flex'; // Подписываемся на события if (opw.on) { opw.on('disconnect', disconnectWallet); opw.on('accountsChanged', (accs) => { if (!accs || accs.length === 0) { disconnectWallet(); } }); } // Загружаем данные await Promise.all([ fetchBalance(), refreshRound(), refreshMyBet() ]); if (aiOn) runAI(); console.log('✅ Wallet connected:', addr); } catch (e) { console.error('connectWallet error:', e); btn.innerHTML = originalText; btn.disabled = false; btn.onclick = connectWallet; let errorMsg = e.message; if (errorMsg.includes('rejected')) { errorMsg = 'Подключение отклонено пользователем'; } else if (errorMsg.includes('Timeout')) { errorMsg = 'Превышено время ожидания. Перезагрузите страницу.'; } alert('❌ Ошибка подключения:\n' + errorMsg); } } // ─── ПРОВЕРКА RPC СОЕДИНЕНИЯ ─────────────────────────────────────────────── async function testRPCConnection() { const testUrls = [ 'https://testnet.opnet.org/api/v1/json-rpc', 'https://testnet.opnet.org/json-rpc', ]; for (const url of testUrls) { try { const controller = new AbortController(); const timeout = setTimeout(() => controller.abort(), 5000); const response = await fetch(url, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ jsonrpc: '2.0', id: 1, method: 'btc_blockCount', params: [] }), signal: controller.signal }); clearTimeout(timeout); if (response.ok) { workingRpc = url; console.log('✅ RPC working:', url); return true; } } catch (e) { console.warn('RPC failed:', url, e.message); } } throw new Error('RPC endpoints unavailable. Check your internet connection.'); } // ─── УЛУЧШЕННАЯ ФУНКЦИЯ getOPWallet ──────────────────────────────────────── function getOPWallet() { // Проверяем window.opnet (OPWallet) if (window.opnet) { console.log('✓ Found window.opnet'); return window.opnet; } // Проверяем Unisat с OP_NET if (window.unisat) { if (window.unisat.opnet) { console.log('✓ Found window.unisat.opnet'); return window.unisat.opnet; } // Проверяем, поддерживает ли Unisat OP_NET if (window.unisat.network === 'testnet' || window.unisat.network === 'regtest') { console.log('✓ Using window.unisat for OP_NET'); return window.unisat; } } console.warn('✗ OPWallet not found'); return null; } // ─── ИСПРАВЛЕННЫЙ rpcRead С ЛУЧШЕЙ ОБРАБОТКОЙ ОШИБОК ─────────────────────── async function rpcRead(sig, args = [], from = null) { const cd = mkCallHex(sig, ...args); const addr = window._contractOverride || getContract(); // Метод 1: OPWallet provider if (window.opnet?.provider?.call) { try { const result = await window.opnet.provider.call({ to: addr, data: cd, }); const d = (result || '').replace(/^0x/, ''); if (d) { workingRpc = '__opnet_provider__'; return d; } } catch(e) { console.warn('[RPC opnet.provider failed]', e.message); } } // Метод 2: Direct fetch с retry const addrs = [addr, CONTRACT_BCRT]; const frm = from || wallet?.address || null; const urls = workingRpc ? [a, cd, frm] : [a, cd]; try { const controller = new AbortController(); const timeout = setTimeout(() => controller.abort(), 8000); const resp = await fetch(url, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ jsonrpc: '2.0', id: 1, method: 'btc_call', params }), signal: controller.signal }); clearTimeout(timeout); if (!resp.ok) continue; const j = await resp.json(); if (j.error) continue; let d = ''; if (typeof j.result === 'string') d = j.result; else if (j.result?.result) d = j.result.result; else if (j.result?.data) d = j.result.data; d = d.replace(/^0x/, ''); if (d) { workingRpc = url; return d; } } catch(e) { console.warn('[RPC FAIL]', url.slice(0, 50), e.message); } } } throw new Error(`Contract call failed: ${sig}.
